Funchat API

Funchat API is a REST based API for Funchat, a very simple single-channel messaging system that is used in Gofore AngularJS training.

Host

Use the following host:

http://funchat-api.herokuapp.com/

REST API Collections

Messages Collection [/messages]

List all Messages

GET /messages
GET /messages?limit=10

Create a Message

POST /messages

Send message as JSON with following structure:

{
    author: 'John',
    content: 'My content'
}

Get Message

GET /messages/{id}

Update Message

PUT /messages/{id}

Use same JSON structure as in create message

Delete Message

DELETE /messages/{id}
